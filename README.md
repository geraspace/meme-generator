# Meme-generator
## Just a fun project I did.
_### Tools: HTML, CSS and JavaScript/React.JS._

**Website here: https://geraspace.github.io/meme-generator/**

In this module I used the React.JS **useState** hook and the event listeners **onChange** and **onClick** to re-render the website and show new meme templates, and to use the form inputs to show the text on the image. I used the JavaScript **map()** method to import the meme links from an external file.
